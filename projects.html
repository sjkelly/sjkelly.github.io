textcad

textcad was started after many hours of developing deeply-parametric designs in OpenSCAD. It is in active development.

HaytchBot

HaytchBot is a deeply-parametric H-Bot platform. It was designed in textcad.

Magpie

Magpie is a comprehensive hardware library for OpenSCAD. Development has stopped to focus on more versitile CAD platforms, however the project remains highly useful.

RepRap Lewis

A deeply-parametric 3D printer prototype. It uses a SCARA system for X-Y motion and a sarrus linkage for Z motion, leading it to be one of the lowest hardware count 3D printers that has never been documented.

Eagle to OpenSCAD

This is an Eagle plug-in to generate OpenSCAD files.

Mixtape Mendel

This was one of my first explorations with parametric design. It never printed more than a few tests. It was my first real foray into Git and hardware design.
